## 코드

```python
import sys

num = int(sys.stdin.readline())
while num>0:
    stack=[]
    str = list(sys.stdin.readline())
    for i in str:
        if i=='(':
            stack.append(i)
        elif i==')':
            if stack:
                stack.pop()
            else:
                print("NO")
                break
    else:
        if not stack:
            print('YES')
        else:
            print('NO')
    num-=1
```
