## 코드

```python
import sys,math

arr = [True for i in range(1000001)]

for i in range(2, 1001):
    if arr[i]:
        for j in range(i+i, 1000001, i):
            arr[j] = False

while True:
    num = int(sys.stdin.readline())
    flag=1
    
    if num==0:
        break

    for i in range(3, num):
        if arr[i] and arr[num-i]:
            print(num, '=', i, '+', num-i)
            flag=0
            break
    
    if flag:
        print('Goldbach\'s conjecture is wrong.')
```
