## 코드
### 첫번째 시도 (런타임 에러-TypeError)
```python
import sys

num = int(sys.stdin.readline())
for i in num:
    a,b=map(int, sys.stdin.readline().split())
    min=min(a,b)
    max=max(a,b)

    for j in range(1,min+1):
        if b%i==0 and a%i==0:
            gcd = j
    min = min/gcd
    max = max/gcd
    print(int(min*max*gcd))
```
