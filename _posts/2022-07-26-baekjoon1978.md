## 2️⃣ 코드

```python
import sys,math

num = int(sys.stdin.readline())
str = list(map(int,sys.stdin.readline().split()))
n = 0

for i in range(num):
    flag = 1

    if str[i]==1:
        flag=0
    
    for j in range(2, int(math.sqrt(str[i])+1)):
        if str[i]%j==0:
            flag=0
            break
    
    if flag:
        n+=1

print(n)
```
