## 코드
### 첫번째 시도 (시간 초과)
```python
import sys

a,b = map(int, sys.stdin.readline().split())
min = min(a,b)
max = max(a,b)

for i in range(min, 0, -1):
    if(a%i==0 and b%i==0):
        print(i)
        break

for i in range(max,a*b+1):
    if(i%a==0 and i%b==0):
        print(i)
        break
```
<br/>

### 두번째 시도
```python
import sys

a,b = map(int, sys.stdin.readline().split())
min = min(a,b)
max = max(a,b)

for i in range(1, min+1):
    if b%i==0 and a%i==0:
        gcd = i

print(gcd)
min = min/gcd
max = max/gcd
print(int(min*max*gcd))
```
