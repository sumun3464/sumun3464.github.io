## 2️⃣ 코드

```python
def sum_n(x):
    if x==1:
        return 1
    elif x==2:
        return 2
    elif x==3:
        return 4
    else:
        return sum_n(x-1)+sum_n(x-2)+sum_n(x-3)

n=int(input())

for i in range(n):
    num = int(input())
    num = sum_n(num)
    print(num)
```
