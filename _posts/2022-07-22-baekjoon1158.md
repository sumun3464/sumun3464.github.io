## 2️⃣ 코드

```python
import sys

n,k = map(int, sys.stdin.readline().split())
table = list(range(1,n+1))
result = []
num=k-1

for i in range(n):
    if len(table) > num:
        result.append(table.pop(num))
        num+=k-1
    else:
        num = num%len(table)
        result.append(table.pop(num))
        num+=k-1

print('<',', '.join(str(i) for i in result), ">", sep='')
```
